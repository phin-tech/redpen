# Diff View Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add three-mode diff viewing (split, unified, highlights) to Red Pen's code review interface, allowing reviewers to see what changed between any two git refs.

**Architecture:** Rust backend computes diffs via the `similar` crate and serves structured hunk data to the Svelte frontend. The frontend renders diffs through CodeMirror 6 decorations (highlights/unified modes) or paired CodeMirror instances (split mode). A new `DiffEditor.svelte` component accepts content as props, unlike the existing `Editor.svelte` which reads from a global store.

**Tech Stack:** Rust (`similar`, `git2`), Svelte 5 (runes), CodeMirror 6 (decorations, gutters, widget decorations), Tauri 2 commands.

**Spec:** `docs/superpowers/specs/2026-03-24-diff-view-design.md`

```rust
use git2::Repository;
use similar::{Algorithm, TextDiff};
use std::path::Path;

fn get_file_at_ref(repo: &Repository, file_path: &str, git_ref: &str) -> Result<String, String> {
    if git_ref == "working-tree" {
        let workdir = repo.workdir().ok_or("Bare repository")?;
        let full_path = workdir.join(file_path);
        std::fs::read_to_string(&full_path)
            .map_err(|e| format!("Cannot read file: {}", e))
    } else {
        let reference = if git_ref == "HEAD" {
            repo.head().map_err(|e| format!("Cannot resolve HEAD: {}", e))?
        } else {
            repo.find_branch(git_ref, git2::BranchType::Local)
                .map(|b| b.into_reference())
                .or_else(|_| repo.find_reference(&format!("refs/tags/{}", git_ref)))
                .or_else(|_| repo.find_reference(&format!("refs/heads/{}", git_ref)))
                .map_err(|_| format!("Cannot resolve ref: {}", git_ref))?
        };

        let commit = reference.peel_to_commit()
            .or_else(|_| {
                let oid = git2::Oid::from_str(git_ref)
                    .map_err(|e| format!("Invalid ref '{}': {}", git_ref, e))?;
                repo.find_commit(oid)
                    .map_err(|e| format!("Cannot find commit: {}", e))
            })
            .map_err(|e| format!("Cannot resolve to commit: {}", e))?;

        let tree = commit.tree().map_err(|e| format!("Cannot get tree: {}", e))?;
        let entry = tree.get_path(Path::new(file_path))
            .map_err(|_| format!("File '{}' not found at ref '{}'", file_path, git_ref))?;
        let blob = repo.find_blob(entry.id())
            .map_err(|e| format!("Cannot read blob: {}", e))?;
        let content = std::str::from_utf8(blob.content())
            .map_err(|_| "File appears to be binary".to_string())?;
        Ok(content.to_string())
    }
}
```

```rust
#[tauri::command]
pub fn compute_diff(
    directory: String,
    file_path: String,
    base_ref: String,
    target_ref: String,
    algorithm: String,
) -> Result<DiffResult, String> {
    let repo = Repository::discover(&directory)
        .map_err(|e| format!("Not a git repository: {}", e))?;

    let repo_root = repo.workdir().ok_or("Bare repository")?.to_path_buf();
    let abs_path = Path::new(&file_path);
    let rel_path = abs_path.strip_prefix(&repo_root)
        .unwrap_or(Path::new(&file_path));
    let rel_path_str = rel_path.to_str().ok_or("Invalid file path")?;

    let old_content = get_file_at_ref(&repo, rel_path_str, &base_ref).unwrap_or_default();
    let new_content = get_file_at_ref(&repo, rel_path_str, &target_ref).unwrap_or_default();

    let algo = match algorithm.as_str() {
        "myers" => Algorithm::Myers,
        _ => Algorithm::Patience,
    };

    let diff = TextDiff::configure()
        .algorithm(algo)
        .diff_lines(&old_content, &new_content);

    let mut hunks = Vec::new();
    for group in diff.grouped_ops(3) {
        let mut changes = Vec::new();
        let mut old_start = u32::MAX;
        let mut new_start = u32::MAX;
        let mut old_count = 0u32;
        let mut new_count = 0u32;

        for op in &group {
            match op {
                similar::DiffOp::Equal { old_index, new_index, len } => {
                    if old_start == u32::MAX { old_start = *old_index as u32 + 1; }
                    if new_start == u32::MAX { new_start = *new_index as u32 + 1; }
                    for i in 0..*len {
                        changes.push(DiffChange {
                            kind: ChangeKind::Equal,
                            old_line: Some((*old_index + i) as u32 + 1),
                            new_line: Some((*new_index + i) as u32 + 1),
                            content: diff.old_slices()[*old_index + i].to_string(),
                        });
                        old_count += 1;
                        new_count += 1;
                    }
                }
                similar::DiffOp::Delete { old_index, old_len, .. } => {
                    if old_start == u32::MAX { old_start = *old_index as u32 + 1; }
                    if new_start == u32::MAX { new_start = 0; }
                    for i in 0..*old_len {
                        changes.push(DiffChange {
                            kind: ChangeKind::Delete,
                            old_line: Some((*old_index + i) as u32 + 1),
                            new_line: None,
                            content: diff.old_slices()[*old_index + i].to_string(),
                        });
                        old_count += 1;
                    }
                }
                similar::DiffOp::Insert { new_index, new_len, .. } => {
                    if old_start == u32::MAX { old_start = 0; }
                    if new_start == u32::MAX { new_start = *new_index as u32 + 1; }
                    for i in 0..*new_len {
                        changes.push(DiffChange {
                            kind: ChangeKind::Insert,
                            old_line: None,
                            new_line: Some((*new_index + i) as u32 + 1),
                            content: diff.new_slices()[*new_index + i].to_string(),
                        });
                        new_count += 1;
                    }
                }
                similar::DiffOp::Replace { old_index, old_len, new_index, new_len } => {
                    if old_start == u32::MAX { old_start = *old_index as u32 + 1; }
                    if new_start == u32::MAX { new_start = *new_index as u32 + 1; }
                    for i in 0..*old_len {
                        changes.push(DiffChange {
                            kind: ChangeKind::Delete,
                            old_line: Some((*old_index + i) as u32 + 1),
                            new_line: None,
                            content: diff.old_slices()[*old_index + i].to_string(),
                        });
                        old_count += 1;
                    }
                    for i in 0..*new_len {
                        changes.push(DiffChange {
                            kind: ChangeKind::Insert,
                            old_line: None,
                            new_line: Some((*new_index + i) as u32 + 1),
                            content: diff.new_slices()[*new_index + i].to_string(),
                        });
                        new_count += 1;
                    }
                }
            }
        }

        if old_start == u32::MAX { old_start = 1; }
        if new_start == u32::MAX { new_start = 1; }

        hunks.push(DiffHunk { old_start, old_count, new_start, new_count, changes });
    }

    Ok(DiffResult { base_ref, target_ref, hunks, old_content, new_content })
}
```

```rust
#[tauri::command]
pub fn list_refs(directory: String) -> Result<RefList, String> {
    let repo = Repository::discover(&directory)
        .map_err(|e| format!("Not a git repository: {}", e))?;

    let head = repo.head().ok();
    let head_name = head.as_ref().and_then(|h| h.shorthand().map(|s| s.to_string()));

    let mut branches = Vec::new();
    for branch_result in repo.branches(Some(git2::BranchType::Local))
        .map_err(|e| format!("Cannot list branches: {}", e))?
    {
        let (branch, _) = branch_result.map_err(|e| format!("Branch error: {}", e))?;
        if let Some(name) = branch.name().ok().flatten() {
            branches.push(BranchInfo {
                is_current: head_name.as_deref() == Some(name),
                name: name.to_string(),
            });
        }
    }
    branches.sort_by(|a, b| b.is_current.cmp(&a.is_current).then(a.name.cmp(&b.name)));

    let mut tags = Vec::new();
    let _ = repo.tag_foreach(|_oid, name_bytes| {
        if let Ok(name) = std::str::from_utf8(name_bytes) {
            let short = name.strip_prefix("refs/tags/").unwrap_or(name);
            tags.push(short.to_string());
        }
        true
    });

    let mut recent_commits = Vec::new();
    if let Ok(mut revwalk) = repo.revwalk() {
        let _ = revwalk.push_head();
        revwalk.set_sorting(git2::Sort::TIME).ok();
        for oid_result in revwalk.take(10) {
            if let Ok(oid) = oid_result {
                if let Ok(commit) = repo.find_commit(oid) {
                    recent_commits.push(CommitInfo {
                        sha: oid.to_string()[..7].to_string(),
                        short_message: commit.summary().unwrap_or("").to_string(),
                    });
                }
            }
        }
    }

    Ok(RefList { branches, tags, recent_commits })
}
```

```typescript
import { invoke } from "@tauri-apps/api/core";
import type { DiffResult, DiffMode, RefList } from "$lib/types";

interface DiffState {
    enabled: boolean;
    mode: DiffMode;
    baseRef: string;
    targetRef: string;
    diffResult: DiffResult | null;
    algorithm: "patience" | "myers";
    loading: boolean;
    error: string | null;
}

let state = $state<DiffState>({
    enabled: false,
    mode: "highlights",
    baseRef: "HEAD",
    targetRef: "working-tree",
    diffResult: null,
    algorithm: "patience",
    loading: false,
    error: null,
});

let refs = $state<RefList | null>(null);

export function getDiffState() {
    return state;
}

export function getRefs(): RefList | null {
    return refs;
}

export async function enterDiff(directory: string, filePath: string) {
    state.enabled = true;
    state.error = null;
    await loadRefs(directory);
    await computeDiff(directory, filePath);
}

export function exitDiff() {
    state.enabled = false;
    state.diffResult = null;
    state.error = null;
    state.loading = false;
}

export function setDiffMode(mode: DiffMode) {
    state.mode = mode;
}

export async function setDiffRefs(directory: string, filePath: string, base: string, target: string) {
    state.baseRef = base;
    state.targetRef = target;
    await computeDiff(directory, filePath);
}

export async function swapRefs(directory: string, filePath: string) {
    const temp = state.baseRef;
    state.baseRef = state.targetRef;
    state.targetRef = temp;
    await computeDiff(directory, filePath);
}

export async function computeDiff(directory: string, filePath: string) {
    state.loading = true;
    state.error = null;
    try {
        const result = await invoke<DiffResult>("compute_diff", {
            directory,
            filePath,
            baseRef: state.baseRef,
            targetRef: state.targetRef,
            algorithm: state.algorithm,
        });
        state.diffResult = result;
    } catch (e) {
        state.error = e instanceof Error ? e.message : String(e);
        state.diffResult = null;
    } finally {
        state.loading = false;
    }
}

export async function loadRefs(directory: string) {
    try {
        refs = await invoke<RefList>("list_refs", { directory });
    } catch {
        refs = null;
    }
}

export function setAlgorithm(algo: "patience" | "myers") {
    state.algorithm = algo;
}
```

```typescript
    eq(other: GhostLineWidget) { return this.count === other.count; }

// --- Highlights mode ---

export function highlightsModeExtensions(diffResult: DiffResult): Extension {
    const insertedLines = new Set<number>();
    for (const hunk of diffResult.hunks) {
        for (const change of hunk.changes) {
            if (change.kind === "insert" && change.newLine != null) {
                insertedLines.add(change.newLine);
            }
        }
    }

    return ViewPlugin.fromClass(
        class {
            decorations: DecorationSet;
            constructor(view: EditorView) {
                this.decorations = this.buildDecorations(view);
            }
            update(update: ViewUpdate) {
                if (update.docChanged || update.viewportChanged) {
                    this.decorations = this.buildDecorations(update.view);
                }
            }
            buildDecorations(view: EditorView): DecorationSet {
                const builder = new RangeSetBuilder<Decoration>();
                const sorted = Array.from(insertedLines).sort((a, b) => a - b);
                for (const lineNum of sorted) {
                    if (lineNum <= view.state.doc.lines) {
                        const line = view.state.doc.line(lineNum);
                        builder.add(line.from, line.from, addedLine);
                    }
                }
                return builder.finish();
            }
        },
        { decorations: (v) => v.decorations }
    );
}

// --- Unified mode ---

export interface UnifiedDocResult {
    syntheticDoc: string;
    lineMap: Map<number, number>; // synthetic line -> new_content line
    extensions: Extension;
}

export function buildUnifiedDocument(diffResult: DiffResult): UnifiedDocResult {
    const lines: string[] = [];
    const lineMap = new Map<number, number>();
    const lineTypes: Array<"equal" | "insert" | "delete"> = [];
    const oldLineNums: Array<number | null> = [];
    const newLineNums: Array<number | null> = [];

    let syntheticLine = 0;
    for (const hunk of diffResult.hunks) {
        for (const change of hunk.changes) {
            syntheticLine++;
            lines.push(change.content.replace(/\n$/, ""));
            lineTypes.push(change.kind);
            oldLineNums.push(change.oldLine);
            newLineNums.push(change.newLine);
            if (change.kind !== "delete" && change.newLine != null) {
                lineMap.set(syntheticLine, change.newLine);
            }
        }
    }

    const syntheticDoc = lines.join("\n");

    // Line highlight decorations
    const highlightPlugin = ViewPlugin.fromClass(
        class {
            decorations: DecorationSet;
            constructor(view: EditorView) {
                this.decorations = this.buildDecorations(view);
            }
            update(update: ViewUpdate) {
                if (update.docChanged) {
                    this.decorations = this.buildDecorations(update.view);
                }
            }
            buildDecorations(view: EditorView): DecorationSet {
                const builder = new RangeSetBuilder<Decoration>();
                for (let i = 0; i < lineTypes.length; i++) {
                    const lineNum = i + 1;
                    if (lineNum <= view.state.doc.lines) {
                        const line = view.state.doc.line(lineNum);
                        if (lineTypes[i] === "insert") {
                            builder.add(line.from, line.from, addedLine);
                        } else if (lineTypes[i] === "delete") {
                            builder.add(line.from, line.from, removedLine);
                        }
                    }
                }
                return builder.finish();
            }
        },
        { decorations: (v) => v.decorations }
    );

    // Dual gutter: old line numbers (left column)
    const oldGutter = gutter({
        class: "cm-diff-old-gutter",
        lineMarker: (_view, line) => {
            const lineNum = _view.state.doc.lineAt(line.from).number;
            const idx = lineNum - 1;
            if (idx < oldLineNums.length) {
                return new DiffLineNumber(oldLineNums[idx]);
            }
            return null;
        },
    });

    // Dual gutter: new line numbers (right column)
    const newGutter = gutter({
        class: "cm-diff-new-gutter",
        lineMarker: (_view, line) => {
            const lineNum = _view.state.doc.lineAt(line.from).number;
            const idx = lineNum - 1;
            if (idx < newLineNums.length) {
                return new DiffLineNumber(newLineNums[idx]);
            }
            return null;
        },
    });

    return {
        syntheticDoc,
        lineMap,
        extensions: [oldGutter, newGutter, highlightPlugin],
    };
}

// --- Split mode ---

export interface SplitDecorationResult {
    oldExtensions: Extension;
    newExtensions: Extension;
}

export function buildSplitDecorations(diffResult: DiffResult): SplitDecorationResult {
    const deletedLines = new Set<number>();
    const insertedLines = new Set<number>();

    // Build ghost line map: for each hunk, calculate padding needed on each side
    // ghostsOld[lineNum] = number of ghost lines to insert AFTER lineNum in old doc
    // ghostsNew[lineNum] = number of ghost lines to insert AFTER lineNum in new doc
    const ghostsOld = new Map<number, number>();
    const ghostsNew = new Map<number, number>();

    for (const hunk of diffResult.hunks) {
        let deletes = 0;
        let inserts = 0;
        let lastDeleteLine = 0;
        let lastInsertLine = 0;

        for (const change of hunk.changes) {
            if (change.kind === "delete" && change.oldLine != null) {
                deletedLines.add(change.oldLine);
                deletes++;
                lastDeleteLine = change.oldLine;
            } else if (change.kind === "insert" && change.newLine != null) {
                insertedLines.add(change.newLine);
                inserts++;
                lastInsertLine = change.newLine;
            }
        }

        // The side with fewer changed lines gets ghost padding
        if (inserts > deletes && lastDeleteLine > 0) {
            ghostsOld.set(lastDeleteLine, inserts - deletes);
        } else if (deletes > inserts && lastInsertLine > 0) {
            ghostsNew.set(lastInsertLine, deletes - inserts);
        }
    }

    const oldExtensions = ViewPlugin.fromClass(
        class {
            decorations: DecorationSet;
            constructor(view: EditorView) { this.decorations = this.build(view); }
            update(u: ViewUpdate) { if (u.docChanged) this.decorations = this.build(u.view); }
            build(view: EditorView): DecorationSet {
                const builder = new RangeSetBuilder<Decoration>();
                const lines = Array.from(deletedLines).sort((a, b) => a - b);
                for (const lineNum of lines) {
                    if (lineNum <= view.state.doc.lines) {
                        const line = view.state.doc.line(lineNum);
                        builder.add(line.from, line.from, removedLine);
                    }
                }
                // Add ghost lines
                for (const [afterLine, count] of Array.from(ghostsOld.entries()).sort((a, b) => a[0] - b[0])) {
                    if (afterLine <= view.state.doc.lines) {
                        const line = view.state.doc.line(afterLine);
                        builder.add(line.to, line.to, Decoration.widget({
                            widget: new GhostLineWidget(count),
                            side: 1,
                            block: true,
                        }));
                    }
                }
                return builder.finish();
            }
        },
        { decorations: (v) => v.decorations }
    );

    const newExtensions = ViewPlugin.fromClass(
        class {
            decorations: DecorationSet;
            constructor(view: EditorView) { this.decorations = this.build(view); }
            update(u: ViewUpdate) { if (u.docChanged) this.decorations = this.build(u.view); }
            build(view: EditorView): DecorationSet {
                const builder = new RangeSetBuilder<Decoration>();
                const lines = Array.from(insertedLines).sort((a, b) => a - b);
                for (const lineNum of lines) {
                    if (lineNum <= view.state.doc.lines) {
                        const line = view.state.doc.line(lineNum);
                        builder.add(line.from, line.from, addedLine);
                    }
                }
                for (const [afterLine, count] of Array.from(ghostsNew.entries()).sort((a, b) => a[0] - b[0])) {
                    if (afterLine <= view.state.doc.lines) {
                        const line = view.state.doc.line(afterLine);
                        builder.add(line.to, line.to, Decoration.widget({
                            widget: new GhostLineWidget(count),
                            side: 1,
                            block: true,
                        }));
                    }
                }
                return builder.finish();
            }
        },
        { decorations: (v) => v.decorations }
    );

    return { oldExtensions, newExtensions };
}

// --- Scroll sync for split mode ---
// Uses proportional scrolling: maps the scroll percentage from source to target.
// With ghost lines (widget decorations), each side has a different total scroll height,
// so percentage-based sync keeps corresponding hunks roughly aligned.
// A more precise hunk-based mapping would require tracking line-to-line correspondence
// from the DiffResult, but proportional sync is simpler and works well in practice
// since ghost lines compensate for the line count differences.

export function scrollSync(leftView: EditorView, rightView: EditorView): () => void {
    let syncing = false;

    function syncScroll(source: EditorView, target: EditorView) {
        if (syncing) return;
        syncing = true;

        const sourceDOM = source.scrollDOM;
        const targetDOM = target.scrollDOM;
        const sourceMax = sourceDOM.scrollHeight - sourceDOM.clientHeight;
        const targetMax = targetDOM.scrollHeight - targetDOM.clientHeight;

        if (sourceMax > 0 && targetMax > 0) {
            const ratio = sourceDOM.scrollTop / sourceMax;
            targetDOM.scrollTop = ratio * targetMax;
        }

        syncing = false;
    }

    const onLeftScroll = () => syncScroll(leftView, rightView);
    const onRightScroll = () => syncScroll(rightView, leftView);

    leftView.scrollDOM.addEventListener("scroll", onLeftScroll);
    rightView.scrollDOM.addEventListener("scroll", onRightScroll);

    return () => {
        leftView.scrollDOM.removeEventListener("scroll", onLeftScroll);
        rightView.scrollDOM.removeEventListener("scroll", onRightScroll);
    };
}
```

## Task 12: Handle Annotation Blocking in Unified Mode

**Files:**
- Modify: `src/components/EditorPane.svelte` (or `src/components/DiffEditor.svelte`)

In unified mode, delete lines should not be annotatable. The `lineMap` from `buildUnifiedDocument` maps synthetic lines to real new-content lines. Lines not in the map are delete lines.

- [ ] **Step 1: Add selection filtering for unified mode**

When unified mode is active and the user selects text, intercept the selection and check if the selected line(s) are in the `lineMap`. If any selected line is a delete line (not in the map), block the annotation creation.

In `DiffEditor.svelte`, add an `onSelectionChange` prop and wire it to the CodeMirror update listener. In `EditorPane.svelte`, when in unified mode, wrap the parent's `onSelectionChange` with a filter:

```typescript
function handleUnifiedSelection(fromLine: number, fromCol: number, toLine: number, toCol: number) {
    // Check if all selected lines are in the lineMap (annotatable)
    for (let line = fromLine; line <= toLine; line++) {
        if (!unifiedLineMap.has(line)) {
            // Selection includes a delete line — block annotation
            return;
        }
    }
    // Map synthetic line numbers to real new-content line numbers
    const mappedFromLine = unifiedLineMap.get(fromLine)!;
    const mappedToLine = unifiedLineMap.get(toLine)!;
    onSelectionChange?.(mappedFromLine, fromCol, mappedToLine, toCol);
}
```
